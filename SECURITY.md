# Security Policy

## Supported Versions

python: 3.12.0 version


## Reporting a Vulnerability

Sometimees bugs, may occur, or imports may load wrong, or code may not execute as intended.

In such a case, You should provide;

1. Which file is the bug situated in.
2. Description of bug.
3. Steps to Recreate bug.
4. (Optional) Possible resolution code.

You may also copy the problematic file to your solutions folder and resolve it in yout fork, proceeding which it can be merged to main by maintainers, as users cannot directly write outside their solutions folder.
